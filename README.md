Finished at : *****

## Installation
#### Web Application
``cd app && npm install``
#### API
``cd api && npm install``

## Run project
To run API and webApp at the same time use
``npm start``
in root folder
#### Web Application & API
To run webApp and API separately go to correct folder and run
``cd api && npm start``
